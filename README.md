# Blahaj Jousting
because the sharks yearn for violence

dual smol rc robots that drive around with smol blahajs on top. you win by smacking the opponents haj off of their car. colon three.

all customizeable at runtime, operates over esp-now with a full field management system, central status lighting, score tracking, and much more. the codebase here is so complicated it could probably operate an entire FRC event with pretty minimal changes lol.

this insanity has a full object oriented API built on top of somewhere around a kertrillion different layers of abstraction and insanity.
